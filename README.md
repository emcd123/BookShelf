Installation
------------

Is not hosted online curretly therefore to run the site you need to use localhost and the following technologies.


Full LAMP stack needed

Apache Web Server https://httpd.apache.org/
MySQL https://www.mysql.com/
PHP 7 http://php.net/

Also required:
  Bootstrap
  jQuery
  Adminer
